COLA AQUI SUAS IMAGENS: Substitua os arquivos *_placeholder.jpg dentro de assets/img/ pelos seus arquivos reais.
Exemplos de nomes recomendados:
 - destaque.png (350x320) -> para a seção de destaque
 - foto-perfil.jpg -> sua foto principal
 - galeria-1.jpg, galeria-2.jpg, etc.
 - video-thumb-1.jpg, video-thumb-2.jpg -> miniaturas para vídeos

Mantenha os nomes ou atualize os caminhos no HTML.
